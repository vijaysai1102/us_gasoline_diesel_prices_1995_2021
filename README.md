This project analyzes the dataset “U.S. Gasoline and Diesel Retail Prices (1995–2021)” to explore long-term fuel price trends, historical patterns, and variations over time.

The project combines Python, Tableau, and D3.js to create both static and interactive data visualizations. These visualizations are presented through a multi-page website, allowing users to explore insights through charts, dashboards, and interactive elements.

## Dataset
Name: U.S. Gasoline and Diesel Retail Prices (1995–2021)
Time Period: 1995 – 2021
Description: Historical retail gasoline and diesel price data across the United States
Purpose: Analyze long-term trends, price fluctuations, and comparative fuel behavior

## Tools & Technologies
Python – Data cleaning, preprocessing, and chart generation
Tableau – Static and exploratory visualizations
D3.js – Interactive and dynamic visualizations
HTML / CSS / JavaScript – Website development and styling
